PWA ICONS NEEDED

Please create and place the following icon files in this public/ folder:

1. icon-192.png (192x192 pixels)
2. icon-512.png (512x512 pixels)

These icons are required for the Progressive Web App (PWA) to work properly.

QUICK OPTION - Create Simple Icons:
1. Go to https://www.pwabuilder.com/imageGenerator
2. Upload any logo or image
3. Generate icons
4. Download and place them here

OR USE THESE FREE TOOLS:
- RealFaviconGenerator: https://realfavicongenerator.net/
- Favicon.io: https://favicon.io/
- Canva: Create a simple design with "FlashCards" text

DESIGN SUGGESTIONS:
- Use the app's primary color (blue: #3b82f6)
- Add the app name "FlashCards"
- Keep it simple and recognizable
- Ensure good contrast for visibility
